## How this site works

Fun fact: this entire site is static — no frameworks, no build step. Everything you see is rendered from a single `config.js` file, and blog posts like this one are plain markdown files.

```js
// Publishing a post is just this:
{
  slug: "my-new-post",
  title: "My New Post",
  date: "2026-06-12",
  file: "posts/my-new-post.md",
}
```

> Simple tools, used well, go a long way.
